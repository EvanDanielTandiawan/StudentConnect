# CampusLink 🎓

CampusLink is a Flutter-based mobile application designed to help university students connect with each other based on their study program, interests, and academic background.

This project was developed as part of a university project and focuses on creating a clean and interactive mobile UI using Flutter.

---

## ✨ Features

* 🔐 Login system
* 👤 Student profile page
* 📝 Edit profile functionality
* 🤝 Add friend feature
* 📚 Program/Lab information
* 💾 Local data storage using Shared Preferences
* 📱 Modern Material Design UI

---

## 🛠️ Tech Stack

* Flutter
* Dart
* Shared Preferences
* Material Design

## 🚀 Getting Started

### 1. Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/CampusLink.git
```

### 2. Open Project

```bash
cd CampusLink
```

### 3. Install Dependencies

```bash
flutter pub get
```

### 4. Run Application

```bash
flutter run
```
